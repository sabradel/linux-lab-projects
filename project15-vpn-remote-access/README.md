# Secure Remote Access VPN Server with OpenVPN + Wazuh

## Project Goal
Build a secure VPN server on an existing Ubuntu Wazuh server to allow Kali Linux and Windows clients to connect securely. Monitor VPN activity using Wazuh.

## Lab Architecture
- Ubuntu Server: OpenVPN + Wazuh
- Kali Linux: VPN Client
- Windows: VPN Client
