# Project 29: Bare Metal Wazuh SOC Lab

## Objective
Deploy Wazuh SIEM on a bare-metal Ubuntu server and use it to detect and investigate security events.

## Environment
- Server: Ubuntu 22.04 (Bare Metal)
- Access: SSH via Tailscale
- SIEM: Wazuh
- Endpoints: To be added (Windows + Linux)
- Attacker: Kali Linux

## Project Workflow
This project is built and documented step-by-step:
- Each action is captured with screenshots
- Commands and notes are recorded
- Detection scenarios are tested and analyzed
